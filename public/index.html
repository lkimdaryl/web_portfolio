Selected Work

Restaurant Management System

The Restaurant Management System is a dynamic web application I developed as part of an engineering class project. Built using FastAPI and MySQL, this system streamlines restaurant operations by facilitating menu item management and order processing. Its interface is user-friendly and leverages RESTful APIs to manage menu items and orders efficiently. Furthermore, I integrated some error handling mechanisms to enhance the application's reliability.

SecureBand

SecureBand is a group project developed in one of my classes. It is an IoT GPS tracking device designed to aid parents and caretakers in monitoring the locations of their children. Combining hardware and software components, the project utilizes an ESP32 microcontroller for GPS data transmission to an external MQTT server. On the software side, a FastAPI-based web server, alongside JavaScript, HTML, and CSS, powers the user interface, offering functionalities such as account registration, login, settings management, child device tracking, and emergency contact information access. The communication between hardware and software relies on MQTT, with data stored in a MySQL database. There are some limitations we encountered but the project represents a viable first iteration with future optimization potential and the aspiration of expanding into mobile applications for broader accessibility.
